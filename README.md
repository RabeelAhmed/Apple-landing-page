3D Apple Landing Page:
This project is a 3D landing page inspired by Apple, created using React, React Three Fiber, React Drey, and styled with Tailwind CSS. It is part of my ongoing journey in learning and mastering the MERN stack.

Technologies Used:
React: A JavaScript library for building user interfaces.
React Three Fiber: A React renderer for Three.js, which allows for 3D graphics in React.
React Drey: A library for 3D components in React.
Tailwind CSS: A utility-first CSS framework for styling.
Features:
3D animations and graphics.
Responsive design with Tailwind CSS.
Modern, sleek interface inspired by Apple.
